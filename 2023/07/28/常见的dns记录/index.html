常见的dns记录

发表于 2023-07-28

这里列举一下常见的dns记录，做域名解析会用到。

  1. A记录：访问时显示访问ipv4的内容
  2. AAAA记录：访问时显示访问ipv6的内容
  3. CNAME记录：访问时显示访问另一个域名的内容
  4. NS记录：给子域单独设置另一个名称服务器
  5. MX记录：设置邮箱服务器,优先使用优先级较小的服务器
举例

  1. dobug.eu.org 通过A记录指向 172.168.9.44 表示访问“dobug.eu.org/*”时页面显示172.168.9.44/*，但网址显示dobug.eu.org

  2. dobug.eu.org 通过AAAA记录指向 1080: 0: 0: 0: 8: 800: 200C : 417A 表示访问“dobug.eu.org/*”时页面显示1080: 0: 0: 0: 8: 800: 200C : 417A/*，但网址显示dobug.eu.org

  3. dobug.eu.org 通过CNAME记录指向 baidu.com 表示访问“dobug.eu.org/*”时页面显示baidu.com/*，但网址显示dobug.eu.org

  4. abc.dobug.eu.org 通过NS记录指向 kellen.ns.cloudflare.com 表示abc.dobug.eu.org的名称服务器中有一项是kellen.ns.cloudflare.com

  5. dobug.eu.org 通过MX记录指向 mxdomain.qq.com 表示dobug.eu.org的邮箱服务器中有一项是mxdomain.qq.com
